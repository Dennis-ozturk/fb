Book System V.1.7Alpha API Docs

To be able to do requests with our API, you need to register and login to generate your API key at http://face-saving-compres.000webhostapp.com/

will always need have your API_KEY in the URL. "?api=API_KEY"

Post request

Info: Id is auto increment therefore we only need to post name and location from the client. And in the URL we need to insert the wanted "TABLE_NAME" and our "API_KEY"

Request Write this Input Data
POST http://face-saving-compres.000webhostapp.com/request.php?TABLE_NAME?api=API_KEY

{
 "name": "Penguin Random House",
 "location": "New York, USA"
}

Get request

Get all publishers

Info: to get the data from all publishers, we simply use the same url as before but with a get request instead.

Get by id

Info: to get the data from a specific publisher by their ID, we have to add a "/" after the TABLE_NAME in the URL and then type the ID of the publisher we want to get.

Request Write this Returned Data
GET http://face-saving-compres.000webhostapp.com/request.php?TABLE_NAME?api=API_KEY

{
 "example_field_id": 2,
 "example_field1": "example_field_value",
 "example_field2": "example_field_value"
},
{
 "example_field_id": 3,
 "example_field1": "example_field_value",
 "example_field2": "example_field_value"
},
...

GET http://face-saving-compres.000webhostapp.com/request.php?TABLE_NAME/ID?api=API_KEY

{
 "example_field_id": 3,
 "example_field1": "example_field_value",
 "example_field2": "example_field_value"
}

Put request

Info: to update the data we will need to specify the data we want to change by entering the ID to the url. We have to add a "/" to the url after the TABLE_NAME and then type the ID we want to update. We also have to send the new data via put request from the client. This data is the one we want to overwrite the existing data to.

Request Write this Input Data
PUT http://face-saving-compres.000webhostapp.com/request.php?TABLE_NAME/ID?api=API_KEY

{
 "name": "new name",
 "location": "new location"
}

Delete request

Info: to delete data we will need to specify the TABLE_NAME we want to delete from by inserting it's ID to the URL. We have to add a "/" after the TABLE_NAME and then type the ID of the we want to delete.

Request Write this
DELETE http://face-saving-compres.000webhostapp.com/request.php?TABLE_NAME/ID?api=API_KEY
