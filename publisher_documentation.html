Publisher

Post request

URL: http://localhost/git/PHP/wies18-integration/grupparbete/fb/request.php?publisher

Info: Id is auto increment therefore we only need to insert name and location data.

example data:

{
 "name": "Penguin Random House",
 "location": "New York, USA"
}

Get request

Get all publishers

URL: http://localhost/git/PHP/wies18-integration/grupparbete/fb/request.php?publisher

Info: to get the data from all publishers, we simply use the same url as before but with a get request.

Get by id

Example URL: http://localhost/git/PHP/wies18-integration/grupparbete/fb/request.php?publisher/3

Info: to get the data from a specific publisher by their id, we have to add a "/" to the url and then type the id of the publisher we want to get.

Put request

Example URL: http://localhost/git/PHP/wies18-integration/grupparbete/fb/request.php?publisher/3

Info: to update the data we will need to specify the data we want to change by entering the publisher id to the url. We have to add a "/" to the url and then type the id of the publisher we want to update. We also have to insert the new data that we want to change to.

example data:

{
 "name": "Changed name",
 "location": "Changed location"
}

Delete request

Example URL: http://localhost/git/PHP/wies18-integration/grupparbete/fb/request.php?publisher/3

Info: to delete a publisher we will need to specify the publisher we want to delete by inserting it's id to the url. We have to add a "/" to the url and then type the id of the publisher we want to delete.
